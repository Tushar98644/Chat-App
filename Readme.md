## Chat-app

This is a chat app built with Next.js, Redux, Node.js, Express, and Socket.io.

## Getting Started

First, run the development server:

```bash
pnpm run dev
# or
yarn dev
# or
npm run dev
```

Open [http://localhost:8000](http://localhost:3000) with your browser to see the result.
